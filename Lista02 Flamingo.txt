9) A JVM é uma máquina virtual que executa programas Java. Ela interpreta o bytecode Java, gerenciando memória, coleta de lixo e garantindo a portabilidade dos programas entre diferentes sistemas operacionais.

10) Projeto refere-se a um conjunto organizado de arquivos, pastas e recursos que compõem um software específico. Um projeto pode incluir o código-fonte, bibliotecas, arquivos de configuração, imagens e outros elementos necessários para o desenvolvimento de uma aplicação ou sistema de software

11) Pacote é uma maneira de organizar e estruturar classes e interfaces relacionadas. Ele agrupa esses elementos em um namespace específico, fornecendo uma hierarquia lógica e evitando conflitos de nomes. Um pacote é uma coleção de classes e interfaces que estão relacionadas entre si.

12) Classe é um bloco de código que encapsula dados e comportamentos relacionados. Ela é um dos conceitos fundamentais da programação orientada a objetos. As classes servem como modelos para criar objetos, que são instâncias dessas classes.
